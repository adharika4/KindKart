# KindKart

KindKart is a resale and donation web platform inspired by OLX, where users can sell old products or buy products to donate directly to NGOs.

This project was built as a hackathon prototype using AI-assisted tools.

## Features
- User login and signup
- Role-based access (User, NGO, Delivery Partner, Admin)
- Product listing with image upload
- AI-based price estimation (simulated)
- Price range slider for fair pricing
- Marketplace to browse products
- Direct donation to NGOs
- Anonymous donation option
- NGO requirement board
- Pickup and delivery tracking
- OTP-based delivery confirmation
- In-app chat interface
- AI chatbot assistant (simulated)
- Impact credits and donation certificates
- Admin dashboard for verification and moderation
- Safety and emergency support during delivery
